# Abbess — Champion Summary

## Executive Summary
- **Role:** ATK Nuker / Decrease DEF
- **Archetype:** AoE Nuker / Debuff Specialist
- **Best Content:** Dragon, Fire Knight, Arena
- **Booking ROI:** High
- **Investment Notes:** Top-tier for AoE nuking and DEF Down. Excels in Arena and Dungeons.
- **Synergy/Relentless Viability:** Not recommended; skill set does not benefit from extra turns

## Recommended Gear & Stats
- **Primary Stat:** ATK
- **Focus Stats (Arena/Dungeons):** ATK, C.DMG, SPD, ACC
- **Focus Stats (Clan Boss):** ATK, C.RATE, SPD, ACC
- **Gear Sets (PvP Offense):** Savage, Cruel, Crit Damage
- **Gear Sets (PvP Defense):** Immunity, Stoneskin
- **Gear Sets (Clan Boss):** Lifesteal, Perception
- **Gear Sets (Dungeons):** Savage, Cruel
- **Gear Sets (Hydra):** Perception, Speed
- **Gear Sets (Iron Twins):** Accuracy, Speed
- **Gear Sets (Solo Farming):** Lifesteal, Speed
- **Gear Tradeoffs:**
    - Savage: Pros: Maximizes ignore DEF for nuking; Cons: No survivability

## Skill & Attack Order
- **Ideal Skill Order:** A2 -> A3 -> A1
- **Recommended Attack Order:** A2 -> A3 -> A1
- **First Turn Skill:** Mass Impalement
- **Disabled Skills:** None

### Skill Details
- **Arrow of Rebuke**: Attacks 1 enemy. Removes 1 random debuff from this Champion and places it on the target. (Type: Single Target, Cooldown: None, Multiplier: 2.8x ATK)
- **Mass Impalement**: Attacks all enemies. 75% chance of placing a 60% Decrease DEF debuff for 2 turns. (Type: AOE, Cooldown: 4 turns, Multiplier: 5.1x ATK)
- **Divine Wrath**: Attacks all enemies. Damage is proportional to either this Champion’s ATK or the target’s DEF or ATK stat, whichever is highest. Deals an extra 30% C.DMG if the target’s ATK is highest. Ignores 30% of the target’s DEF if the target’s DEF is highest. (Type: AOE, Cooldown: 6 turns, Multiplier: 4.5x ATK (scales with target's highest stat))

## Mastery & Booking
- **Best Mastery (PvE/PvP):** Helmsmasher
- **Best Mastery (Clan Boss):** Warmaster
- **Booking Impact:** High — Books increase damage, debuff chance, and reduce cooldowns.
