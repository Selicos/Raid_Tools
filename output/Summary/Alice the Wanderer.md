# Alice the Wanderer — Champion Summary

## Executive Summary
- **Role:** ATK Nuker / Cooldown Control
- **Archetype:** Single-Target Nuker / AoE Cooldown Manipulator
- **Best Content:** Fire Knight, Eternal Dragon
- **Booking ROI:** High
- **Investment Notes:** Top-tier for cooldown control and single-target nuking. Excels in Arena, Eternal Dragon, and Fire Knight.
- **Synergy/Relentless Viability:** Not recommended; skill set does not benefit from extra turns

## Recommended Gear & Stats
- **Primary Stat:** ATK
- **Focus Stats (Arena/Dungeons):** ATK, C.DMG, SPD, ACC
- **Focus Stats (Clan Boss):** ATK, C.RATE, SPD, ACC
- **Gear Sets (PvP Offense):** Savage, Lethal, Merciless
- **Gear Sets (PvP Defense):** Stoneskin, Untouchable
- **Gear Sets (Clan Boss):** Lifesteal, Perception
- **Gear Sets (Dungeons):** Savage, Cruel
- **Gear Sets (Hydra):** Perception, Speed
- **Gear Sets (Iron Twins):** Perception, Speed
- **Gear Sets (Solo Farming):** Lifesteal, Speed
- **Gear Tradeoffs:**
    - Savage: Pros: Maximizes ignore DEF for nuking; Cons: No survivability

## Skill & Attack Order
- **Ideal Skill Order:** A2 -> A3 -> A1
- **Recommended Attack Order:** A2 -> A3 -> A1
- **First Turn Skill:** Clockwork Cyclone
- **Disabled Skills:** None

### Skill Details
- **Vorpal Sword**: Attacks 1 enemy 2 times. Places an extra hit if the target has any active skills on cooldown. Each hit has an 80% chance of decreasing the duration of a random buff on the target by 1 turn. (Type: Single Target (multi-hit), Cooldown: None, Multiplier: 1.7x ATK per hit)
- **Clockwork Cyclone**: Attacks all enemies. 75% chance to increase the cooldowns of all enemy skills by 2 turns. 75% chance to decrease each target’s Turn Meter by 15% (or 30% if a skill is maxed). (Type: AOE, Cooldown: 4 turns, Multiplier: 4.2x ATK)
- **Queenslayer**: Attacks 1 enemy, ignores 20% of the target’s DEF. If the initial target is killed, also attacks the enemy with the lowest HP (ignores 20% DEF again). (Type: Single Target (chain), Cooldown: 4 turns, Multiplier: 5x ATK)
- **Passive:** [Passive] Whenever this Champion attacks, inflicts 3% bonus damage to each target for each turn remaining on all of their skill’s cooldowns. [Active] Resets the cooldown of one of this Champion’s skills each time they kill an enemy (once per skill).

## Mastery & Booking
- **Best Mastery (PvE/PvP):** Helmsmasher
- **Best Mastery (Clan Boss):** Warmaster
- **Booking Impact:** High — Books increase damage, debuff chance, and reduce cooldowns.
