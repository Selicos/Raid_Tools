### 1.1 Speed Tune & Basics

**Speed Tune Type:** 3:2 (3 champion turns per 2 boss turns)  
**Speed Range:** 267-277 SPD (tight 10 SPD band)  
**Boss SPD:** 170 (post-turn 1)  
**Ratio:** 1.57x (267√∑170) to 1.63x (277√∑170)

**Critical Rules:**
- ‚úÖ All champions 263-280 SPD (allows ¬±3 SPD variance)
- ‚úÖ SPD boots on ALL champions (+45 base, or +40 with SPD ascension substat if rare)
- ‚úÖ Speed set 2pc on ALL champions (+12% base SPD)

### 1.4 Usage Guide

**Turn 1 Priority:**
1. **Stag A2** ‚Üí Dec DEF/ATK (must land first for team damage amp)
2. **Geomancer A3** ‚Üí HP Burn + Weaken (damage core)
3. **Brogni A3** ‚Üí Block Debuffs (survivability)
4. **Mithrala A2** ‚Üí Shields (not A3, save for cleanse gaps)
5. **Aniri A1** ‚Üí Build to A2 for buff extension turn 4

**Turn 4 (First Gap Turn):**
- **Aniri A2** ‚Üí Extend Block Debuffs (2 turns ‚Üí 3 turns)
- **Mithrala A3** ‚Üí Cleanse if debuffs landed during gap

**Turn 7 (Second Gap Turn):**
- Repeat Aniri A2 extension if available
- Mithrala A3 cleanse as needed

**Stun Target Management:**
- **Geomancer** is slowest (267 SPD) = always stunned
- 4.6k DEF = stun damage reduced by ~75% (vs 2k DEF baseline)
- Boss stun damage: ~30k (reduced to ~7.5k with 4.6k DEF)
- Geomancer 55k HP = can tank 7+ stuns before danger

**Boss Rotation Tracking:**
- Affinity rotates daily: Force ‚Üí Magic ‚Üí Spirit ‚Üí Void (4-day cycle)
- Check in-game Clan Boss tab for today's affinity
- **Void days:** Use all 4 daily keys (max damage priority)
- **Spirit days:** Use keys (68M verified), or save for Void if close to 72M unlock

**Affinity Strategy:**

| Boss | Strategy | Expected Damage |
|------|----------|-----------------|
| **Spirit** üü¢ | Geo weak affinity, Brogni strong | 60-68M (verified) |
| **Force** üü° | Brogni weak, Stag strong | 55-62M |
| **Magic** üü° | Stag weak, Geo strong | 52-60M |
| **Void** ‚≠ê | All neutral (BEST) | **65-76M** (target 72M+) |

**Auto vs Manual:**
- **Auto:** 90%+ success, excellent AI (all champions prioritize correctly)
- **Manual:** +5-8% damage (perfect skill timing, no wasted turns)
- **Recommendation:** Auto unless pushing for 72M unlock (manual Void boss)

**Optimization Checklist:**
- ‚úÖ Geomancer ACC 280+ verified
- ‚úÖ All champions 263-280 SPD verified
- ‚ö†Ô∏è Mithrala 618 ACC overkill (regear priority: Perception ‚Üí Regeneration)
