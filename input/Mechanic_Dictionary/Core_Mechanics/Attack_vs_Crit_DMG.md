## This document has reference charts for scaling types if damage depending on base stat used for the skill/etc calculation. 
- It shows charts for scaling the base state like attack or defense and then crit damage, in intervals based on the likely stat spread. 
- Use these charts to recommend stat/gear builds in guides and reviews. 
# Most damage charts this equation:
- Note that attacks that scale off Attack and Defense are greater than 1, like 5x or 6.6x, while HP based attacks are often below 1, like 0.2x.
 
$$
\text{Total Damage} = \frac{\text{STAT}}{1000} \cdot \left( 1 + \frac{\text{CD}\%}{100} \right)
$$

Some attacks scale off max oss HP and will use the same equation as A, using the max boss HP per boss mechanics for damage cal

# other general guidelines for damage and stat setup. Validate the assertions before using (one time)
- If you EVER have less than 40% chance to crit it is always better to take an equivalent amount of crit rate than crit damage (equivalent in this case meaning the 1.3 CD to CR ratio).

- If you already have 78% EXTRA crit damage from gear, it is ALWAYS more efficient to get crit rate instead of crit damage assuming an equivalent gain. 

You can use this table to decide if a piece of gear is more damage or not based on crit alone.

People keep asking me about attack and if this helps them decide. It does not. Every champ and every attack they have scales differently with attack damage as far as I know. So there is no way for me to calculate the value of attack vs crit rate unless you go to stage 1 kaerok castle and do an experiment to see how your abilties scale with attack points. If you do that I could then figure out when it's optimal to stop taking attack. (and so could you)

It should be noted that this is just a blanket rule based on the calculation. Obviously if you have champs with abilities with increased crit chances or skills with extra crit damage built in they would not follow these "rules". Every rule has an exception.
