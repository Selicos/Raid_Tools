# Champion Intake Queue

## Purpose
This file is the intake queue for new champions to be processed through the champion scraper. Add champion names here (one per line) to batch process them.

## Usage
1. Add champion names to the list below (one per line)
2. Run: `python Tools/champion_scraper/champion_scraper.py --list input/Champion_Intake_list.md`
3. Champions will be scraped, validated, and added to Champion_stats.md with JSON entries created
4. After successful processing, remove champions from this list
